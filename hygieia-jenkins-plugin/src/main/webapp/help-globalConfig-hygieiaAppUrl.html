Specify the URL to Hygieia UI Application(s), e.g. http://hygieia.yourcompany.com.

This field if populated will print Hygieia dashboard url(if available) on the Jenkins build console if publish build data option is selected.

If more than one Hygieia Server URLs are configured, you may configure the same number of Hygieia UI URLs e.g. http://hygieia.yourcompany.com/env1, http://hygieia.yourcompany.com/env2

The values of this should be in conjuction with the order in which Hygieia Server URLs are configured and should match the api in the order configured.

For the below configuration :
Hygieia Server URL - http://hygieia.yourcompany.com/api,http://hygieia.yourcompany.com/api2
Hygieia UI URL - http://hygieia.yourcompany.com/env1, http://hygieia.yourcompany.com/env2
UI served by http://hygieia.yourcompany.com/env1 should be using http://hygieia.yourcompany.com/api1 and
UI served by http://hygieia.yourcompany.com/env2 should be using http://hygieia.yourcompany.com/api2
